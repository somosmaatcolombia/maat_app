# MAAT — Proyecto para Claude Code

## Setup rápido (3 pasos)

1. **Descomprime** el ZIP
2. **Copia** tu `maat_dashboard.html` a `src/` (el mentor ya está incluido)
3. **Ejecuta** `cd maat-project && claude`

Claude Code leerá `CLAUDE.md` automáticamente — tiene todo el contexto:
esquema DB, reglas, patrones, vistas, métricas.

## Verificar estructura

```bash
bash setup.sh
```
